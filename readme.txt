The ultimate CRM and management system for WordPress agencies. Manage clients, hosting expiries, maintenance contracts in one place.


== Description ==

**MioWeb Agency Management CRM** is the all-in-one solution built for web agencies, freelancers, and WordPress professionals who need to organize their business workflow. Stop using messy spreadsheets to track your clients' hosting expiries or maintenance renewals.

Whether you are a solo developer or a growing agency, this plugin provides a centralized hub to monitor your entire ecosystem: from technical hosting details to recurring maintenance billing cycles.

= Key Business Benefits =
* **Never Miss a Renewal**: Track hosting and maintenance expiries with an intuitive dashboard.
* **Centralized Client Data**: A dedicated CRM to store VAT numbers, contacts, and addresses.
* **Project Portfolio**: Catalog every website, plugin, and theme you've ever built.
* **Data Sovereignty**: Your data stays in *your* database. No external SaaS fees.

= Features =

* **Client Management** - Store all client information including contacts, VAT numbers, addresses
* **Hosting Management Expiry Guard** - Track hosting plans, providers, expiry dates, costs, and credentials
* **Maintenance Contracts** - Manage recurring maintenance contracts with renewal dates and amounts
* **Website Catalog** - Keep a database of websites built, with screenshots and technical details
* **Plugin Repository** - Track WordPress plugins developed, versions, and repository status
* **Theme Collection** - Manage themes with framework info, parent themes, and screenshots
* **Interactive Dashboard** - Overview of all activities with upcoming renewals and statistics
* **Advanced Filtering** - Search and filter across all sections
* **Advanced Client CRM** – Comprehensive profiles for all your clients with custom metadata.
* **Developer Portfolio (CPT)** – Organize your work: Websites (with screenshots), custom Plugins, and Themes.
* **Smart Dashboard** – Real-time statistics and "Alert" widgets for upcoming deadlines..

= Why MioWeb Agency? =

Managing a web agency involves juggling multiple clients, hosting accounts, maintenance contracts, and development projects. This plugin brings everything together in one place, integrated directly into your WordPress admin.

= Use Cases =

* **Freelancers** - Keep track of all your clients and their hosting renewals
* **Small Agencies** - Manage multiple team members' projects and client relationships
* **Development Shops** - Catalog all plugins and themes you've developed for easy reference

== Installation ==

1. Upload the `mio-web-agency` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access the plugin via the new 'MioWeb' menu in your admin sidebar
4. Start adding your clients, hosting plans, and maintenance contracts

== Frequently Asked Questions ==

= Do I need coding knowledge to use this plugin? =

No, the plugin is designed to be used by anyone. All features are accessible through a user-friendly admin interface.

= Can I customize the fields? =

The current version includes all standard fields for agency management. Custom fields support is planned for future updates.

= Is the plugin translation ready? =

Yes, all strings are translatable. The plugin comes with full translation support.

= Where is my data stored? =

All data is stored in your WordPress database. Clients, hosting, and maintenance use custom tables, while websites, plugins, and themes use custom post types.

= Is this a full CRM for WordPress? =

Yes, it functions as a lightweight, specialized CRM tailored specifically for the needs of web developers and digital agencies.

= How does the expiry tracking work? =

You enter the renewal dates for hosting and maintenance. The dashboard will automatically highlight contracts that are approaching their expiry date.

= Is my client data safe? =

Absolutely. Unlike SaaS solutions, MioWeb stores everything locally on your server. We have no access to your data.
